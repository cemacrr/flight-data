Flight Data

Flight data is from the OpenSky Network. The API can sometimes be slow to respond, so it may take a while for the data to load and update.

Data is shown for flights within a 100km radius of Manchester airport, and is updated every 20 seconds (if the OpenSky API responds in time).

A random color is picked for each flight, and the circle size is set based on altitude, with previous markers for each flight reduced in size each time the data is updated. Markers older than 60 seconds are removed with each update.

For more information about how the emissions data is calculated, etc., please see the README on GitHub. It may be completely incorrect ...
